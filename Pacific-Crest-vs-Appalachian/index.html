Pacific Crest Trail vs. Appalachian Trail


Link to high resolution map

Created by Sean Stewart @ Ronin Geospatial

The map was created by using the Georeferencer tool in QGIS and placed over a Raisz_Landforms1957 image file which is used as the basemap. This a comparison between the Pacific Crest Trail in the west and the Appalachian Trail in the east. Which trail would you choose to hike? Can you tell which would be more difficult?

  • Appalachian Trail Conservancy
